# System

## Functions

| Function | Description |
| -------- | ----------- |
| `shouldQuit()` | Check if the application should quit. |

## Detailed Documentation

### shouldQuit

Check if the application should quit.

**Returns:**

| Type | Description |
| ---- | ----------- |
| `boolean` | True if the application should quit. |

### sleep

Sleep for the specified number of milliseconds.

**Parameters:**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `ms` | `integer` | The number of milliseconds to sleep. |
